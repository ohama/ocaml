Ocamllex Tutorial

Ocamllex Adaptation: SooHyoung Oh

This is a tutorial on how to use ocamllex which is distributed with Ocaml language.

This document borrowed lots of part from flex manual.

Please mail all comments and suggestions to <ohama100 at gmail dot com>

The latest version can be found at https://ohama.github.io/ocaml/ocamllex-ocamlyacc/ocamllex-tutorial/index.html.

The companion tutorial for ocamlyacc is available at https://ohama.github.io/ocaml/ocamllex-ocamlyacc/ocamlyacc-tutorial/index.html.

You can find the source of this document in ocamllex-tutorial-src. For printing, pdf is presented.

You can download the source of examples used in this document from ocamllex-examples directory.

This document is just a re-formatting of the old one. There have been many improvements since then in OCaml programming language but that is not reflected in this document.
